## Challange otorgado por Esto Es.

Para realizarlo se utilizó React.
Se realizó la configuración desde cero, de webpack y de babel.
Al ser un desafío solo de front, se utilizó Redux como store global (redux-toolkits), 
en donde estarán los usuarios y los projectos persistidos, esto se logra gracias a redux-persist.
Para tener usuarios en el store se utilizó mockaroo.

Para darle estilos, se utilizó bootstrap, reac-bootstrap y CSS puro.
